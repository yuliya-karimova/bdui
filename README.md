# Backend Driven UI Demo

Демонстрационное приложение для презентации по теме "Backend Driven UI".

## Архитектура

- **Backend** (порт 3001) - API для управления контентом страниц
- **BFF** (порт 3002) - Backend for Frontend, адаптирует данные для фронтенда
- **Frontend** (порт 3000) - Публичная часть сайта с динамическими блоками
- **Admin** (порт 3003) - Админ-панель для редактирования страниц

## Запуск

```bash
docker-compose up --build
```
или
```
docker compose up --build
```

После запуска:
- Публичный сайт: http://localhost:3000
- Админ-панель: http://localhost:3003
- Backend API: http://localhost:3001
- BFF API: http://localhost:3002

## Типы блоков

1. **Text Block** - Текстовый блок с заголовком и содержимым
2. **Card Block** - Блок с карточками (изображение, заголовок, описание)
3. **Banner Block** - Баннер с изображением и текстом

## Использование

1. Откройте админ-панель (http://localhost:3003)
2. Создайте новую страницу или отредактируйте существующую
3. Добавьте блоки и настройте их содержимое
4. Сохраните изменения
5. Просмотрите результат на публичном сайте (http://localhost:3000)

## Структура проекта

```
bdui/
├── backend/          # Backend API (Node.js + Express)
├── bff/              # Backend for Frontend
├── frontend/         # Публичный фронтенд (React)
├── admin/            # Админ-панель (React)
└── docker-compose.yml
```

## Особенности

- **Backend Driven UI**: Контент страниц управляется через бэкенд
- **BFF слой**: Оптимизирует и трансформирует данные для фронтенда
- **Динамические блоки**: Три типа блоков (текст, карточки, баннеры)
- **Админ-панель**: Визуальный редактор для управления контентом
- **Docker**: Все сервисы запускаются в контейнерах
